Creating Your Bot:

In-order to make it easy to get started we have sample bots bellow which are already set up and simply need to be uploaded to Heroku or another hosting service. We currenlty only have sample bots for Python and Javascript with more on the way!

Bot Requierments

When creating your bot, there must be an endpoint `/bet`. This is the endpoint that Probot Playground will call in-order to communicate wiht your bot. You are free to handle the request from Probot Playground however you would like, however your bot must respond within five second. Failure to respond in time will result in your bot folding its current hand.

Hosting Your Bot:

You are free to use any hosting service you may like. If you are looking for a good hosting service, we recomend using Heroku. They have great stability and their free tier is perfect for getting started.

Game Data:

The data your bot receives will be a JSON object consisting of a ‘state’ and ‘history’ attribute. The ‘state’ is a JSON object of the current state of the game. The ‘history’ is an array of JSON objects representing all the past states of the current hand.
